## Setup 

- Ruby 3.1.4
- PostgresQL

Install dependencies:

```ruby 
bundle install
```
Create database and run migrations:

```ruby 
bin/rails db:create db:migrate 
```
Server run:

```ruby 
bin/rails server
```
